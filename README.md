# ⚖️ Landing Page Profissional - Hina Pereira de Aguiar

Landing page profissional desenvolvida para apresentar a trajetória acadêmica, habilidades e objetivos de Hina Pereira de Aguiar, estudante de Direito pela UNINASSAU.

O projeto foi desenvolvido com foco em elegância, responsividade e experiência do usuário, utilizando uma identidade visual inspirada na área jurídica através de uma paleta sofisticada composta por preto, branco, cinza e dourado.

## 🔗 Acesse o Projeto

https://ginaaguiar.github.io/landing-page-comercial/

## ✨ Funcionalidades

* Layout moderno e responsivo
* Navegação suave entre seções
* Menu mobile adaptado para dispositivos móveis
* Botão "Voltar ao topo"
* Animações de entrada utilizando Intersection Observer
* Design otimizado para desktop, tablet e smartphone
* Integração com Instagram
* Estrutura organizada e de fácil manutenção
